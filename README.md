# Documentation

## Lucky Store - E-Commerce
E-Commerce Website with 2 combine technologie \
→ **API REST** created with **Node Express** \
→ **Front End** with **React**


## Deployment

To deploy this project run

```bash
  npm i
```

then
```bash
  npm run dev
```
